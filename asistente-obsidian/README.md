# Asistente Obsidian (Premium)

Gestión completa de tu segundo cerebro: Zettelkasten, MOCs, weekly reviews y procesamiento de inbox.

## Cómo usar

1. En AikaOS: **Create Worker** > **Select Folder** > selecciona esta carpeta
2. Empieza a chatear

## Qué puedes pedirle

- "Busca todas mis notas sobre productividad y dime cómo se conectan"
- "Crea una nota sobre [tema que estés investigando]"
- "Hazme la revisión semanal de mi vault"
- "Crea un MOC para organizar mis notas de marketing"
- "Procesa mi inbox: clasifica y organiza lo que tengo pendiente"

## Skills (3)
- **gestion-vault** — Búsqueda, creación, mantenimiento. Estructura PARA, formato con frontmatter, status seed/growing/evergreen
- **resumen-semanal** — Weekly review: actividad, action items, conexiones, patrones, plan semanal
- **zettelkasten-linking** — Notas atómicas, tipos de enlace, MOCs, ciclo de vida de notas

## Commands (5)
- `buscar-notas` — Búsqueda inteligente con resumen y conexiones
- `crear-nota` — Nota completa con frontmatter, wikilinks y preguntas abiertas
- `revision-semanal` — Weekly review con 8 secciones
- `crear-moc` — Map of Content para organizar notas sobre un tema
- `procesar-inbox` — Clasificar, enriquecer, conectar y ubicar notas del inbox

## MCPs recomendados (opcionales)
- **Obsidian** (`obsidian-mcp-server`) — Acceso completo al vault
- **Filesystem** (`@anthropic/filesystem-mcp`) — Alternativa si no usas el plugin REST API (ya incluido)
