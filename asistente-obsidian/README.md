# Asistente Obsidian (Premium)

Gestión completa de tu segundo cerebro: Zettelkasten, MOCs, weekly reviews y procesamiento de inbox.

## Skills (3)
- **gestion-vault** — Búsqueda, creación, mantenimiento. Estructura PARA, formato de notas con frontmatter, status de notas (seed/growing/evergreen)
- **resumen-semanal** — Weekly review completo: actividad, action items, conexiones, patrones, plan semanal
- **zettelkasten-linking** — Notas atómicas, tipos de enlace (apoya/contradice/extiende/ejemplo), MOCs, ciclo de vida de notas

## Commands (5)
- `buscar-notas` — Búsqueda inteligente con resumen, conexiones y sugerencia de MOC
- `crear-nota` — Nota completa con frontmatter, wikilinks, tipo de conexión y preguntas abiertas
- `revision-semanal` — Weekly review con 8 secciones: actividad, proyectos, action items, ideas, conexiones, atención, patrones, foco
- `crear-moc` — Map of Content para organizar notas sobre un tema
- `procesar-inbox` — Clasificar, enriquecer, conectar y ubicar notas del inbox

## MCPs recomendados
- **Obsidian** (`obsidian-mcp-server`) — Acceso completo al vault: leer, escribir, buscar
- **Filesystem** (`@anthropic/filesystem-mcp`) — Alternativa si no usas el plugin REST API

## Cómo probar
1. Copia skills y commands a tu `.opencode/`
2. Conecta el MCP de Obsidian o Filesystem apuntando a tu vault
3. Prueba: "Busca todas mis notas sobre productividad y dime cómo se conectan"
4. Prueba: "Crea una nota sobre [tema que estés investigando]"
5. Prueba: "Hazme la revisión semanal de mi vault"
6. Prueba: "Procesa mi inbox — clasifica y organiza lo que tengo pendiente"
