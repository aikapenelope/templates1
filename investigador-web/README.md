# Investigador Web (Premium)

Investigación profunda con metodología de fact-checking, verificación de fuentes y reportes profesionales.

## Skills (3)
- **investigacion-web** — Metodología de investigación con búsqueda estratégica, fuentes primarias/secundarias, y formatos por profundidad
- **verificacion-fuentes** — Framework SIFT de fact-checking, escala de confiabilidad, detección de red flags
- **sintesis-reportes** — 4 tipos de reporte: brief ejecutivo, investigación completa, comparativa ponderada, monitoreo

## Commands (5)
- `investigar-tema` — Investigación profunda con estrategia de búsqueda y reporte estructurado
- `comparar-opciones` — Comparativa ponderada con scores, riesgos y veredicto
- `resumen-articulo` — Resumen + análisis crítico de artículos web
- `verificar-dato` — Fact-checking con framework SIFT y veredicto
- `monitorear-tema` — Reporte de seguimiento periódico con cambios y tendencias

## MCPs recomendados
- **Firecrawl** (`firecrawl-mcp`) — Scraping web rápido para investigación
- **Brave Search** (`@anthropic/brave-search-mcp`) — Búsquedas web actualizadas
- **Filesystem** (`@anthropic/filesystem-mcp`) — Guardar reportes en archivos

## Cómo probar
1. Copia skills y commands a tu `.opencode/`
2. Prueba: "Investiga el mercado de fintech en Colombia en 2024"
3. Prueba: "Verifica: ¿Es cierto que México es el segundo mercado de e-commerce en LATAM?"
4. Prueba: "Compara Notion vs Obsidian vs Logseq para gestión de conocimiento"
