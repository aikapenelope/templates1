# Investigador Web (Premium)

Investigación profunda con fact-checking, verificación de fuentes y reportes profesionales.

## Cómo usar

1. En AikaOS: **Create Worker** > **Select Folder** > selecciona esta carpeta
2. Empieza a chatear

## Qué puedes pedirle

- "Investiga el mercado de fintech en Colombia en 2024"
- "Verifica: es cierto que México es el segundo mercado de e-commerce en LATAM?"
- "Compara Notion vs Obsidian vs Logseq para gestión de conocimiento"
- "Resume este artículo: [URL]"
- "Monitorea las noticias sobre regulación cripto en LATAM"

## Skills (3)
- **investigacion-web** — Metodología de investigación, fuentes primarias/secundarias, formatos por profundidad
- **verificacion-fuentes** — Framework SIFT de fact-checking, escala de confiabilidad, detección de red flags
- **sintesis-reportes** — 4 tipos de reporte: brief ejecutivo, investigación completa, comparativa, monitoreo

## Commands (5)
- `investigar-tema` — Investigación profunda con reporte estructurado
- `comparar-opciones` — Comparativa ponderada con scores y veredicto
- `resumen-articulo` — Resumen + análisis crítico de artículos web
- `verificar-dato` — Fact-checking con framework SIFT
- `monitorear-tema` — Reporte de seguimiento con cambios y tendencias

## MCPs recomendados (opcionales)
- **Firecrawl** (`firecrawl-mcp`) — Scraping web rápido
- **Brave Search** (`@anthropic/brave-search-mcp`) — Búsquedas web
- **Filesystem** (`@anthropic/filesystem-mcp`) — Guardar reportes (ya incluido)
