# Ventas B2B (Premium)

Pipeline completo: prospección, propuestas, manejo de objeciones y cierre para el mercado LATAM.

## Cómo usar

1. En AikaOS: **Create Worker** > **Select Folder** > selecciona esta carpeta
2. Empieza a chatear

## Qué puedes pedirle

- "Escribe un email frío para el CEO de una fintech en Colombia que acaba de levantar Serie A"
- "El cliente me dijo que está muy caro. Cómo respondo?"
- "Crea una propuesta comercial para mi servicio de consultoría para empresa X"
- "Dame un reporte de mi pipeline" (pega tus datos de deals)
- "Necesito dar seguimiento a un lead que no me responde hace 2 semanas"

## Skills (3)
- **prospeccion-ventas** — ICP, canales LATAM, calificación BANT+MEDDIC, pipeline con etapas, cadencias de seguimiento
- **propuestas-comerciales** — Estructura de propuesta ganadora, 3 opciones de precio, casos de éxito, cotización rápida
- **negociacion-cierre** — 7 objeciones con respuestas palabra por palabra, técnicas de cierre, reglas de negociación LATAM

## Commands (5)
- `crear-propuesta` — Propuesta completa con 3 opciones de precio
- `seguimiento-lead` — Mensaje personalizado por etapa y canal
- `reporte-pipeline` — Análisis con forecast, deals en riesgo, velocidad
- `manejar-objecion` — Respuesta a objeción específica con script listo
- `email-frio` — Email de prospección o cadencia completa de 5-6 emails

## MCPs recomendados (opcionales)
- **Gmail** (`@anthropic/google-gmail-mcp`) — Enviar emails de seguimiento
- **Google Calendar** (`@anthropic/google-calendar-mcp`) — Agendar reuniones
- **Brave Search** (`@anthropic/brave-search-mcp`) — Investigar prospectos
