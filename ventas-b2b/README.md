# Ventas B2B (Premium)

Pipeline completo: prospección, propuestas, manejo de objeciones y cierre para el mercado LATAM.

## Skills (3)
- **prospeccion-ventas** — ICP, canales LATAM (LinkedIn/email/WhatsApp/referidos), calificación BANT+MEDDIC, pipeline con etapas y probabilidades, cadencias de seguimiento, métricas
- **propuestas-comerciales** — Estructura de propuesta ganadora, 3 opciones de precio (ancla psicológica), casos de éxito, cotización rápida
- **negociacion-cierre** — Las 7 objeciones más comunes con respuestas palabra por palabra, técnicas de cierre, reglas de negociación LATAM

## Commands (5)
- `crear-propuesta` — Propuesta completa con 3 opciones de precio y casos de éxito
- `seguimiento-lead` — Mensaje personalizado por etapa y canal, cadencia de 6 touchpoints
- `reporte-pipeline` — Análisis completo: forecast, deals en riesgo, velocidad, win/loss
- `manejar-objecion` — Respuesta a objeción específica con script listo para usar
- `email-frio` — Email de prospección personalizado o cadencia completa de 5-6 emails

## MCPs recomendados
- **Gmail** (`@anthropic/google-gmail-mcp`) — Enviar emails de seguimiento y prospección
- **Google Calendar** (`@anthropic/google-calendar-mcp`) — Agendar reuniones y follow-ups
- **Brave Search** (`@anthropic/brave-search-mcp`) — Investigar prospectos y empresas

## Cómo probar
1. Copia skills y commands a tu `.opencode/`
2. Prueba: "Escribe un email frío para el CEO de una fintech en Colombia que acaba de levantar Serie A"
3. Prueba: "El cliente me dijo que está muy caro. ¿Cómo respondo?"
4. Prueba: "Crea una propuesta para [tu servicio] para [empresa X]"
5. Prueba: "Dame un reporte de mi pipeline" + pega tus datos de deals
