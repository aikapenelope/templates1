---
name: publicar-propiedad
description: Generar textos optimizados para publicar una propiedad en portales y redes
---

Necesito publicar una propiedad y quiero textos optimizados para cada canal.

Dame los datos de la propiedad (o usa una ficha que ya hayamos creado).

Genero versiones para:
- **Portal inmobiliario** (Inmuebles24, ZonaProp, Metrocuadrado, etc.): Descripción completa, SEO-friendly, con todos los datos técnicos
- **Instagram**: Caption para post o carrusel con gancho, hashtags inmobiliarios, y estructura de slides sugerida
- **Facebook Marketplace**: Texto directo con precio y datos clave
- **WhatsApp**: Mensaje corto con foto + datos esenciales para enviar a contactos
- **Story/Reel**: Script de 30 segundos para tour virtual

Cada versión es nativa del canal, no un copy-paste recortado.

Incluyo:
- Orden sugerido de fotos para cada plataforma
- Hashtags inmobiliarios del país
- Horario sugerido de publicación
- CTA específico para cada canal
