---
name: comparar-propiedades
description: Comparar múltiples propiedades lado a lado para tomar decisión
---

Necesito comparar varias propiedades para decidir cuál comprar o rentar.

Dame los datos de cada propiedad (2-5 propiedades). Para cada una necesito:
- Ubicación
- Precio
- Superficie (m²)
- Características (recámaras, baños, estacionamiento)
- Estado y antigüedad
- Cualquier dato adicional que tengas

Genero:
- Tabla comparativa completa con todas las dimensiones
- Precio/m² de cada una (para comparar manzanas con manzanas)
- Pros y contras de cada propiedad
- Si es inversión: Cap Rate y GRM estimado de cada una
- Análisis de ubicación (qué zona tiene mejor conectividad, servicios, plusvalía)
- Ranking con puntaje ponderado
- Recomendación final según tu perfil (¿buscas vivir o invertir?)
