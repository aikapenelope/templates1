---

Ayúdame a crear un post para redes sociales.

Pregúntame:
1. ¿Para qué plataforma? (X/Twitter, LinkedIn, Instagram, TikTok caption, todas)
2. ¿Cuál es el tema o mensaje principal?
3. ¿Qué tono? (profesional, casual, inspirador, provocador, educativo)
4. ¿Hay un CTA? (link, comentar, guardar, compartir, comprar, agendar)
5. ¿Para qué país o audiencia?

Entrego:
- El post listo para copiar y pegar, nativo de la plataforma
- Framework de copywriting usado (AIDA, PAS, BAB) y por qué
- Hashtags relevantes (cantidad correcta para la plataforma)
- Horario sugerido con zona horaria LATAM
- Sugerencia de visual (descripción de imagen o tipo de carrusel)
- 2 variantes alternativas con diferente ángulo

Si pediste "todas", genero una versión nativa para cada plataforma (no un copy-paste recortado).
