Quiero crear un hilo para X/Twitter.

Pregúntame:
1. ¿Sobre qué tema?
2. ¿Cuál es el insight principal o la lección?
3. ¿Tienes datos, experiencia personal o ejemplos concretos?
4. ¿Cuántos tweets? (recomiendo 5-10 para máximo engagement)
5. ¿Hay un CTA? (seguirte, visitar un link, descargar algo)

Entrego un hilo con:
- **Tweet 1**: Gancho irresistible (dato impactante, promesa clara, o pregunta provocadora)
- **Tweets 2-N**: Un punto por tweet, concreto y con valor. Cada tweet funciona solo pero fluye en secuencia.
- **Tweet final**: Resumen + CTA + "Si te sirvió, RT el primer tweet para que llegue a más personas"
- Cada tweet numerado y listo para copiar
- Sugerencia de imagen para el tweet 1 (los hilos con imagen en el primero tienen más alcance)
- Horario sugerido para publicar
