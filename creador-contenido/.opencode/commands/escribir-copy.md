---
name: escribir-copy
description: Escribir copy persuasivo para landing pages, ads o emails de venta
---

Necesito copy que convierta.

Pregúntame:
1. ¿Para qué? (landing page, anuncio de Facebook/Instagram, email de venta, página de producto, bio)
2. ¿Qué vendes? (producto, servicio, curso, evento)
3. ¿Quién es tu cliente ideal? (perfil, dolor principal, qué desea)
4. ¿Cuál es tu diferenciador? (por qué tú y no la competencia)
5. ¿Tienes testimonios o datos de resultados?

Entrego:
- Copy completo usando el framework más efectivo para el caso (AIDA, PAS, BAB)
- Explicación de por qué funciona (la psicología detrás de cada sección)
- Headline principal + 2 alternativas
- CTA principal + variante
- Si es landing page: estructura completa sección por sección
- Si es ad: versión corta (para feed) + versión larga (para copy extenso)
- Si es email: subject line + preview text + cuerpo + CTA
